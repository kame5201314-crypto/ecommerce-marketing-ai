# 🚀 AI 自動化平台 - 快速啟動指南

## ⚡ 三步驟快速開始

### 第一步:安裝與啟動

```bash
# 1. 進入專案目錄
cd ecommerce-marketing-ai

# 2. 安裝依賴 (首次使用)
npm install

# 3. 啟動開發模式
npm run dev
```

開啟瀏覽器前往: `http://localhost:5173`

## 🎯 八大核心模組功能速覽

### 1. 📝 AI 內容生成中心
**一句話說明:** 自動生成商品文案、SEO文章、社群貼文

**快速使用:**
1. 點選「AI 內容生成」
2. 選擇「商品文案」
3. 輸入商品名稱與特色
4. 點擊「開始生成」
5. 複製結果使用

**適合對象:** 電商賣家、自媒體經營者、行銷人員

---

### 2. 📢 行銷自動化中心
**一句話說明:** FB/Google 廣告優化、自動排程發布

**快速使用:**
1. 進入「行銷自動化」
2. 選擇「FB 廣告生成」
3. 輸入商品資訊
4. 自動生成 3-10 組廣告
5. 選擇最佳版本使用

**適合對象:** 廣告投手、社群經營者、行銷團隊

3. 連接 Bot Token
4. 設定自動回覆規則
5. 啟用 AI Bot

**適合對象:** 有大量客服需求的電商、教育、顧問業

## 🎓 使用技巧

### 技巧 1: 提供詳細描述
**差:** "幫我做一個商品文案"
**好:** "為我的藍牙耳機寫一篇商品文案,強調降噪功能與長續航力,目標受眾是25-35歲上班族"

### 技巧 2: 多生成幾個版本
- 每次生成 3-10 個版本
- 挑選最符合需求的
- 可再微調優化

### 技巧 3: 善用虛擬團隊
- 美編需求 → 找 Aria
- 影片需求 → 找 Victor
- 文案需求 → 找 Chloe
- 數據分析 → 找 Diana

### 技巧 4: 建立自動化流程
從簡單開始:
```
新訂單 → LINE 通知
```

逐步增加:
```
新訂單 → LINE 通知 → 更新 Google Sheet → 庫存同步 → 發送感謝信
```
